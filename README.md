# Rouh Perfume – Full Stack App

متجر عطور متكامل مبني بـ **Laravel (API + Filament Admin)** و **React (Frontend)**، مع نشر على استضافة مشتركة (Hostinger).

## 🧱 Tech Stack

- **Backend:** Laravel, MySQL, Filament
- **Frontend:** React, Vite
- **Auth & Admin:** Filament Panel
- **Deployment:** Shared hosting (Hostinger)
